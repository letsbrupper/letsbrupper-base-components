# Project letsbrupper-base-components
This is set of base components using in Letsbrupper.com

# Prerequisite
- Nodejs v10.7.x
- NPM 6.2.x

# Build & Install
- Install: `$ npm install`
- Run styleguide: `$ npm start`
- Build: `$ npm run build`
- Build styleguide: `$ npm run styleguide`
- Check lint: `$ npm run lint`
